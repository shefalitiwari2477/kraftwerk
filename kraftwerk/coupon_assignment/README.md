# Project: Driving Coupon Acceptance Analysis

### Overview
This project analyzes survey data to determine which factors influence a driver's decision to accept a coupon. 

### Key Insights
* **Existing Habits:** The single best predictor of accepting a bar coupon is how often the driver already visits bars.
* **Timing is Everything:** Coffee coupons are 40% more likely to be accepted in the morning than in the evening.
* **Environmental Context:** Drivers with children are significantly less likely to accept coupons for social venues (Bars/Coffee Houses).

### How to Run
1. Clone this repository.
2. Ensure you have `pandas`, `seaborn`, and `jupyter` installed.
3. Open `prompt.ipynb` to see the full analysis.
